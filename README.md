
## Overview
The Appointment System is a simple and user-friendly web-based platform that allows users to schedule, manage, and track appointments efficiently. It is designed to streamline booking processes.
The system supports both User and Administrator roles, offering an intuitive interface for seamless appointment management.
## Features
  ### Users

- Book an appointment

- Track the status of an appointment using their registered email

### Administrator

* Manage users

* Approve or reject or reschedule appointment requests

* View and manage all bookings through the admin dashboard

* Access admin page via:appointment-system/html/admin.html

### System Architecture

- Frontend: HTML, CSS, JavaScript

- No backend
